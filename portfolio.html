Project Details

Smart Inventory System
  • Category Management: Manages product categories for effective organization and search.
  • Inventory Tracking: Monitors inventory levels across multiple warehouses with real-time updates.
  • Order Management: Handles order processing, tracking, and reporting.
  • Supplier Management: Manages supplier information and transactions.

Built using Spring Boot for backend, React.js for the frontend, and SQL for database management.

Library Management System
  • Book Management: Manages book records and their availability.
  • Requested Book Issue: Handles the process of issuing requested books.
  • Issued Book Records: Maintains records of issued books and their return status.
  • Automatic Fine Management: Automatically calculates fines for late returns.

Developed with React.js for the frontend, Spring Boot for backend services, and SQL for database management.

Learning Management System
  • Teacher Roles: Different roles for teachers including creating questions, tests, and assignments.
  • Student Assignments: Tracks assignments and their status for each student.
  • HOD Panels: Allows Head of Department to monitor student and teacher details and track work progress.

Implemented using React.js for the frontend and Spring Boot for backend development.
